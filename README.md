My personal simple kitty configuration.

<!--Instalation-->
## Instalation of kitty.
#### Arch Linux
```bash
# Arch Linux
$ yay -S kitty
$ git clone https://github.com/srcrapi/kitty-configs.git
$ cd kitty-configs
$ cp kitty.conf ~/.config/kitty
```
